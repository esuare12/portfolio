Projects

RiceBallStream

The RiceBallStream project was developed for a coding class with the purpose of showcasing JavaScript skills including form validation, buttons, carousel display, and other functions along with HTML and CSS. My source of inspiration for this project came from another anime streaming platform, which I use pretty often. I have been working on and honing this website as it started as a project from another class a while ago, and I decided to keep working on it. It started as a simple HTML/CSS project, then I proceeded to add the JavaScript functionality like the carousel, form, and game. Finally, this project demonstrates my proficiency in JavaScript fundamentals such as DOM manipulation, event handling, form validation, and interactive random number game.

Bread + Butter

Bread + Butter, one of my favorite projects, was my senior capstone project. The purpose was to help a local bakery with a sudden business surge to help manager their inventory, reduce waste, and boost their sales through a pre-order online system. This concept was somewhat inspired by an app that helps businesses sell their items before discarding them. The website includes an add-to-cart button on all item cards that updates item quantity added to the cart, a cart that displays items with image, price, and quantities, plus a -/+ sign to update quantities and updates subtotal.
